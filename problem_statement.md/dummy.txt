#  Hostel Allotment Booking System

##  Overview

The Hostel Allotment Booking System is a web-based application designed to simplify and automate the process of hostel room allocation for students. It provides an efficient, transparent, and user-friendly platform for both students and administrators to manage hostel bookings, availability, and records.

This system eliminates manual paperwork, reduces errors, and ensures fair allocation of rooms based on predefined rules.

---

##  Features

###  Student Features

* Register and login securely
* View available hostels and rooms
* Apply for hostel allotment
* Track application status
* Receive notifications for approval/rejection

###  Admin Features

* Admin login dashboard
* Add/manage hostels and rooms
* View student applications
* Approve or reject requests
* Allocate rooms based on availability
* Manage student records

##  Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/hostel-booking-system.git
```

2. Navigate to project folder:

```
cd hostel-booking-system
```

3. Install dependencies:

```
npm install
```

4. Run the application:

```
npm start
```

5. Open in browser:

```
http://localhost:3000
```
